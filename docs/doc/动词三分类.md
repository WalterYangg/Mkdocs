# 动词三分类

**所有动词原型都是由う段结尾**

|  \   | **あ段** | **い段** | **う段(动词)** |  え段  | お段 |
| :--: | :------: | :------: | :------------: | :----: | :--: |
| あ行 |    あ    |    い    |       う       |   え   |  お  |
| か行 |    か    |    き    |       く       |   け   |  こ  |
| さ行 |    さ    |    し    |       す       |   せ   |  そ  |
| た行 |    た    |    ち    |       つ       |   て   |  と  |
| な行 |    な    |    に    |       ぬ       |   ね   |  の  |
| は行 |    は    |    ひ    |       ふ       |   へ   |  ほ  |
| ま行 |    ま    |    み    |       む       |   め   |  も  |
| や行 |    や    |  （い）  |       ゆ       | （え） |  よ  |
| ら行 |    ら    |    り    |       る       |   れ   |  ろ  |
| わ行 |    わ    |  （い）  |     （う）     | （え） |  を  |

## 一类动词（5段动词）

### 原型

- 结尾是う段非る的动词，肯定是5段动词
- あ段+る、う段+る、お段+る
- 三个特例

| 动词 |  假名  |
| :--: | :----: |
| 切る |  きる  |
| 帰る | かえる |
| 走る | はしる |

### 动词变形

**将动词结尾的う段变为い段+ます**

| 原型(う段) | 变形(い段+ます) |
| :--------: | :-------------: |
| 書く(かく) |    書きます     |
| 吸う(すう) |    吸います     |
| 切る(きる) |    切ります     |
| 呼ぶ(よぶ) |    呼びます     |

##  二类动词（上一段动词、下一段动词）

###  原型

|    \    |     \      |
| :-----: | :--------: |
| い段+る | 上一段动词 |
| え段+る | 下一段动词 |

### 动词变形

**将动词结尾的る段直接变为ます**

##  三类动词（ カ变、サ变 ）

###  原型

|  \   |  \   |
| :--: | :--: |
| くる | カ变 |
| する | サ变 |

> する可以将名词变为动词，但并非所有名词都能变，只有被标记为`自サ`的名词才可以

### 动词变形

**无规则，死记硬背**

| 原型 |  变形  |
| :--: | :----: |
| くる | きます |
| する | します |
